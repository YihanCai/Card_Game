# 记忆翻牌游戏

一个基于 Web 的经典记忆翻牌配对游戏，使用纯 HTML + CSS + JavaScript 构建。

## 功能特性

- 4 种难度选择：简单（3×4/6对）、中等（4×4/8对）、困难（4×6/12对）、挑战（6×6/18对）
- 卡片翻转 3D 动画
- 翻牌、配对成功/失败、通关音效
- 步数统计与计时器
- 通关后火柴人放礼炮 + 后空翻庆祝动画
- 响应式布局，支持移动端

## 开始游戏

直接在浏览器打开 `index.html` 即可开始。或在线上玩：[GitHub Pages 链接](https://yihancai.github.io/Card_Game/)

## 技术栈

- HTML5
- CSS3（Flexbox、Grid、3D Transform）
- JavaScript（Canvas、Web Audio API）
