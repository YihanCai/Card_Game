# 记忆翻盘游戏 - 项目计划

## 项目概述
基于 Web 的记忆翻牌配对游戏，用户点击卡片翻转，找到两张相同的卡片即可消除，全部消除即为胜利。

## 技术栈
- **HTML5** - 页面结构
- **CSS3** - 样式与动画
- **JavaScript (Vanilla JS)** - 游戏逻辑（无需框架，轻量简洁）
